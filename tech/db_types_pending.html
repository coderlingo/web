Different types of DBs

Different types of DBs(Pending)

There are different types of DBs that are used in a variety of distributed applications these days. There are multiple DBs out there like mongoDB, redis, cassandra, Hbase to name a very few of the popular ones. But these DBs can be clubbed together into broad categories and not every DB suits every problem that a developer might come across in the course of his career. Thus its important to have an understanding of the types of DBs out there and the specific purpose that they serve.

The databases can be broadly classified into the following categories:

  • Relational Database
  • Document-Oriented Database
  • Graph Database
  • Key-Value Database
  • Time Series Database
  • Wide-Column Database

Relational Database

Structured representation of data. Data has attributes and well-defined relation between them. Usually works well for transactional data since such tables ensure strong ACID properties. These databases do not scale well. To extract tricky relations from the database, joins need to be performed which can be an expensive operation. A lot of effort is needed in the initial phases for coming up with a normalized schema for the db.
Examples: MySQL(open source), PostGreSQL(open source)
