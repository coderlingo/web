Important points in DSA problem approaches

  • If you want to make sure that a pair is not counted twice when traversing through an array, populate the array while traversing instead of traversing an already populated array
  • XOR basics : 1 is a flipper, 0 is a mirror. Matching values are nullified and different values are flagged.
  • ">>>" unsigned right shift. Does not populate the MSB in the new bit positions.
  • Modulo operator basics : 1. Addition property : (A + B) mod C = (A mod C + B mod C)mod C
    2. Mod of a -ve number : (-A) mod C = (-A + C)mod C
    2. Multiplication property of modular arithmetic: (A * B) mod C = (A mod C * B mod C) mod C
    3. Exponentiation property: A^B mod C = ( (A mod C)^B ) mod C
  • 1+2+3+...+n = (n+1)*n/2
  • A DAG always has at least 1 topological order
  • char[] arr = st.toCharArray() // where st is any string
  • String top = new String(st,i,n); // where i is the offset, and n is the length
  • HashMap/HashSet can not have array as key. because internally the hashcode() and equals() methods of array uses the reference and not the content of the array.
  • Arrays.sort(nums, i + 1, n); //this sorts the array within the range
  • List <Integer> list1 = Arrays.stream(arr).boxed().collect(Collectors.toList()); // convert arrays to list
