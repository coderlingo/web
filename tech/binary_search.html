Binary Search

Binary search is one of the most simple yet most trouble some algorithm widely used to reduce search time. Usually recursive method is preferred. I generally attribute to the one usecase we were introduced to it for. Which is Finding if an element is present in an array. But I, from using this method and twisting it to solve a new problem, usually end up making the problem far more complex.

A better approach is to make it iterative. The reason being, in htis process we introduce another variables with stored the probable ans in each step. This ofcourse is not applicable for searching for an element in sorted arrays, but most of the other variations of problem statement which need to be solved like this, it becomes much better instead of keeping track of all the edge cases.
